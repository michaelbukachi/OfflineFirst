### OfflineFirst Demo
Demonstration of a sync strategy for offline first applications

**Requirements**
- AS Electric Eel
- Java 11

#### Module of interest
- sync

#### Assumptions
- DI has aleady been set up
- Daos and models have been implemented
- App UI submission logic has been set up

Run tests with
```bash
./gradlew :sync:testDebugUnitTest
```
